# 🏬 Dubai Hills Mall – Interactive Sales Experience

## 🚀 Live Demo
👉 https://mall-sales-deck-pi.vercel.app/

---

## 🧠 Concept
An immersive, cinematic, AI-powered web experience showcasing Dubai Hills Mall as a premium retail destination.

Inspired by Apple-style storytelling and luxury brand presentation.

---

## ⚙️ Tech Stack

- React + TypeScript
- Tailwind CSS
- Framer Motion
- Vite
- Node.js + Express
- Groq API (LLM)

---

## ✨ Features

- 🎬 Cinematic hero with video
- 🧲 Smooth scroll storytelling
- 🧠 AI-powered business insights (Groq)
- 💎 Glassmorphism UI
- ⚡ Cursor glow & micro interactions
- 📊 Section indicators

---

## 🤖 AI Integration

Used Groq (Llama3 70B) to generate:
- Retail insights
- Dining strategy
- Entertainment monetization

Prompt engineered for:
- Business clarity
- Category differentiation
- Premium tone
